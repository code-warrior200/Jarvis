💡 PRO TIPS

  🎤 Better Voice Recognition:
    • Speak clearly and at normal pace
    • Minimize background noise
    • Keep microphone 6 inches away

  ⚡ Performance Tips:
    • Close unused applications
    • Keep window visible for feedback
    • Click Clear when display gets cluttered

  🎯 Efficient Usage:
    • Use partial app names (e.g., "chrome" works)
    • Mix voice and text as needed
    • Click ❓ Help for command syntax

🆘 TROUBLESHOOTING

  GUI won't open:
    $ pip install tk
    $ python run.py

  Import errors:
    $ pip install -r requirements.txt
    $ python verify_setup.py

  Microphone not working:
    • Use text input instead
    • Check Windows Sound Settings
    • Test microphone with another app

  Slow performance:
    • Close other applications
    • Clear the display (🗑️ button)
    • Restart Jarvis
