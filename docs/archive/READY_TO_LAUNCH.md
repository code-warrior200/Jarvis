## 🆘 Quick Troubleshooting

### GUI Won't Open
```bash
# Install tkinter
pip install tk

# Try again
python run.py
```

### Microphone Not Working
- Check Windows Sound Settings
- Use text input instead
- Check microphone is enabled

### Command Not Recognized
- Speak more clearly
- Click ❓ Help for syntax
- Try typing the command

### Slow Performance
- Close other applications
- Use "Clear" button to declutter
- Restart Jarvis

## 💡 Pro Tips

✨ **Better Voice Recognition**
- Speak clearly and at normal pace
- Minimize background noise
- Keep microphone 6 inches away
- Use pause between commands

🎯 **Efficient Usage**
- Use partial app names (e.g., "chrome" works)
- Combine voice and text as needed
- Click Clear when display gets cluttered
- Check Help for command syntax

⚡ **Performance**
- Close unused apps
- Keep window visible for feedback
- Use text if voice is unreliable
- Restart if response time slows
