# 🚀 START HERE - Launch Jarvis GUI

## ⚡ Quick Start (30 seconds)

### Step 1: Install dependencies (first time only)
```bash
python -m pip install -r requirements.txt
```

### Step 2: Launch Jarvis GUI
```bash
python run.py
```

✨ **That's it!** The beautiful GUI will launch instantly.

---

## 🎯 What You Need to Know

### How to Use
1. **Click 🎤 Listen** → Speak your command
2. **Or type** in the text field → Press Enter
3. **See response** instantly in the display

### Example Commands
- "What time is it?"
- "Open notepad"
- "Minimize chrome"
- "Set volume to 50"
- "Weather in London"

### Buttons
| Button | Function |
|--------|----------|
| 🎤 Listen | Start voice input |
| 📤 Send | Execute typed command |
| ❓ Help | Show all commands |
| 🗑️ Clear | Clear display |

## 🆘 Troubleshooting

### GUI won't launch
```bash
# Install required packages
python -m pip install -r requirements.txt

# Or just tk
pip install tk
```

### Microphone not working
- Check Windows Sound Settings
- Type commands instead of speaking
- Try again with clear voice

### Command not recognized
- Speak more clearly
- Click ❓ Help for command syntax
- Try typing instead
