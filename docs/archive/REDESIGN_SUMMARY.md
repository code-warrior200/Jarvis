## 🎤 How to Use

### Voice Input
1. Click **▶ LISTEN** (cyan button)
2. Button turns **◆ LISTENING ◆** (pink)
3. Status bar shows "◆ LISTENING... SPEAK NOW ◆"
4. Speak your command clearly
5. Response appears in green

### Text Input
1. Type in the input field (right panel)
2. Press **Enter** or click **▶ EXECUTE**
3. Response appears in green
4. Type stays in input field for editing

### Example Commands
```
Voice: "What time is it?"
Text:  open notepad
Voice: "Set volume to 50"
Text:  minimize chrome
Voice: "Tell me a joke"
Text:  weather in london
```

## 🆘 Troubleshooting

### GUI Won't Start
```bash
pip install tk
python run.py
```

### Microphone Issues
- Check Windows Sound Settings
- Test microphone with another app
- Use text input instead

### Display Issues
- Resize window to minimum 1200×800
- Click ▶ CLEAR to reset
- Restart application

### Slow Performance
- Close other applications
- Use text instead of voice
- Restart Jarvis
- Check CPU/Memory usage
