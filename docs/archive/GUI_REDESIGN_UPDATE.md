## 🚀 How to Use

### Launch
```bash
python run.py
```

### Voice Input
1. Click **▶ LISTEN** (cyan button)
2. Button changes to **◆ LISTENING ◆** (pink)
3. Speak your command
4. Response appears in green

### Text Input
1. Type in the input field
2. Press **Enter** or click **▶ EXECUTE**
3. Response appears in green

### Example Commands
```
"What time is it?"
"Open notepad"
"Minimize chrome"
"Set volume to 50"
"System info"
"Help"
```

## 💡 Pro Tips

### Voice Input
- Speak clearly at normal pace
- Minimize background noise
- Use partial app names ("chrome" works)
- Say "Help" to see all commands

### Text Input
- Type full commands for better accuracy
- Press Enter to execute
- Clear display periodically for performance

### UI Management
- Resize window as needed (1400×900 recommended)
- Keep window visible for feedback
- Use color coding to identify responses

## 🆘 Troubleshooting

### GUI Won't Start
```bash
pip install tk
python run.py
```

### Microphone Issues
- Check Windows Sound Settings
- Test with another app
- Use text input instead

### Display Issues
- Resize window to 1200×800 minimum
- Click ▶ CLEAR to reset
- Restart application
