💡 USAGE WORKFLOW

1️⃣  LAUNCH APPLICATION
    $ python run.py
    → GUI opens with initialization sequence
    → Status light pulses cyan/magenta
    
2️⃣  CHOOSE INPUT METHOD
    
    Option A: Voice Input
    ├─ Click ▶ LISTEN (cyan button)
    ├─ Button becomes ◆ LISTENING ◆ (pink)
    ├─ Status shows "◆ LISTENING... SPEAK NOW ◆"
    ├─ Speak your command
    └─ Response appears in cyan
    
    Option B: Text Input
    ├─ Click in input field
    ├─ Type your command
    ├─ Press Enter or click ▶ EXECUTE
    └─ Response appears in cyan
    
3️⃣  VIEW RESPONSES
    • System messages: Magenta
    • Your commands: Cyan
    • Jarvis responses: Green
    • Information: Blue
    • Errors: Pink
    • Timestamps on every line
    
4️⃣  MANAGE INTERFACE
    • Use ▶ HELP for commands
    • Use ▶ CLEAR to reset display
    • Use ▶ ANALYZE for system status
    • Use ▶ SHUTDOWN to exit gracefully
