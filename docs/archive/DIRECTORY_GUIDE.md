# Project Directories

This file marks important directories in the project.

## Directory Structure (Professional Organization)

### `/src` - Source Code
Main application code organized by functionality:
- `core/` - Core engine and business logic
- `ui/` - User interface components
- `utils/` - Utility functions and helpers

### `/docs` - Documentation
Comprehensive project documentation:
- User guides
- Developer guides
- API reference
- Architecture documentation

### `/tests` - Test Suite
Automated tests:
- Unit tests
- Integration tests
- GUI tests

### `/scripts` - Utility Scripts
Development and deployment scripts:
- Setup verification
- Build scripts
- Test runners

### `/config` - Configuration
Configuration files:
- Application settings
- Environment variables
- Default configurations

### `/.github` - GitHub Specific
GitHub workflow and templates:
- CI/CD workflows
- Issue templates
- Pull request templates

---

**Note**: This project follows professional Python packaging standards.
